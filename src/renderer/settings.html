1. Choose Chat

Only one integration is active at a time.

2. Configure Telegram

BackPing will accept replies only from the chat that completes setup.

  1. Create a bot with BotFather.
  2. Paste the bot token and save it.
  3. Test the connection.
  4. Open Telegram and send /start to the bot.
  5. Copy the Codex config after the chat is connected.

2. Configure Slack

BackPing uses Socket Mode and routes answers through each question thread.

  1. Create a Slack app and enable Socket Mode.
  2. Open Settings > Basic Information, scroll to App-Level Tokens, then generate an xapp- token with connections:write.
  3. Add bot scopes: chat:write, im:write, im:history.
  4. Subscribe to message.im and enable Interactivity.
  5. Install the app, then paste the tokens and your Slack user ID.
  6. Find your user ID from Slack: Profile > More > Copy member ID.

3. Connect Agents

Use user-level MCP config so BackPing is available across Codex and Claude sessions.
